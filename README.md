# PackMoveGo Desktop

Public frontend application for PackMoveGo.

## Requirements

- Node.js `>=18.14.0`
- npm `>=8.0.0`

## Quick Start

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5001` by default.

## Commands

- `npm run dev` - Start development server
- `npm start` - Build and preview production mode locally

## Contributing

New contributors are welcome.

1. Open or claim an issue
2. Create a branch
3. Submit a pull request with a clear description

Please read `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `SECURITY.md` before contributing.

GitHub issue and pull request templates are included to help keep reports clear and safe.

## Security

Never commit credentials, secrets, or local environment files.
Report vulnerabilities privately as described in `SECURITY.md`.
Use `config/.env.example` as a placeholder template only, never with real secrets.
